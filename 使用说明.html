PDF 论文翻译工具 · 使用说明

把英文论文 PDF 翻译成中文 PDF：图片、公式、表格、排版原样保留， 译文出现在原文对应的位置；参考文献按学术惯例保持原样。支持扫描版（自动 OCR）。 翻译由大模型完成（默认 DeepSeek，也可用本地免费模型）。

一、第一次使用（只需三步）

第 1 步双击 启动PDF翻译.bat。 首次会自动安装所需组件（需联网，几分钟），随后浏览器会自动打开操作界面。
启动时会出现一个黑色小窗口，那是程序本体，请勿关闭；用完后关掉它即退出程序。
第 2 步填入 API Key。到 https://platform.deepseek.com/api_keys 注册并创建一个 Key（形如 sk-…，需充值少量金额），粘贴到界面"API Key"栏， 勾选"记住"。点"测试连接"确认可用。
第 3 步点"添加…"选择 PDF（可多选），点 "▶ 开始翻译"。完成后点"打开 PDF"即可阅读。
建议第一次先把"试译前 N 页"设为 2，花几分钱看看效果， 满意后改回 0 翻整篇——已试译的部分有缓存，不会重复收费。

二、翻译好的文件在哪？

默认生成在原 PDF 的同一个文件夹，文件名规则：

输出模式生成文件名适合什么时候用
纯中文原名_translation.pdf 日常阅读。译文直接替换原文，版式与原文一致。
双语·前后页原名_translation_bilingual.pdf 打印对照。原文页与译文页交替出现。
双语·左右对照原名_translation_sidebyside.pdf 屏幕上核对原文。每页变成宽页：左半原文、右半译文，逐页对齐。

三、界面各选项是什么意思

四、费用与省钱

  • 点开始后，日志会先显示费用预估：「预计需请求 N 段，约 M 万 tokens」。 一篇 15 页论文全文约 3~4 万 tokens，用 DeepSeek 通常只要几毛钱。
  • 翻译缓存：译过的段落自动存在本机 （程序\cache\）。同一篇文档再翻（换输出模式、 补失败段、重新生成）几乎不再花钱。
  • 省钱三招：先试译 2 页再翻整篇；模型换 v4-flash； 取消思考模式。
  • 翻译中途网络出错不会前功尽弃：失败的段落自动保留原文并提示数量， 重新点一次"开始翻译"即可只补失败的部分。

五、扫描版 PDF（图片型）

无需任何设置：检测到某页没有文字层时自动进行本地 OCR 识别（不联网、 不需要显卡），再走正常翻译流程。日志会显示「第 N 页为扫描版，已 OCR 识别」。 每页识别约需几秒，首次使用多约 20 秒初始化。

扫描版的限制：识别质量受扫描清晰度影响；整页倒置（180°）的 扫描件不会自动纠正；译文以白底覆盖回填，深色纸张背景会露出白条。

六、常见问题

提示 SSL 连接错误 / 连接失败？

几乎都是 VPN、加速器或安全软件拦截所致。按顺序试：① 关闭 VPN/加速器； ② 界面勾选直连模式；③ 暂时关闭 360/电脑管家的"HTTPS/SSL 扫描"； ④ 浏览器打开 https://api.deepseek.com 看网络是否通。

提示 401 / 402 / 403？

Key 填错或余额不足。到 DeepSeek 官网检查 Key 与充值。

翻译完发现某些段落还是英文？

三种情况：① 参考文献——有意保留（学术惯例，保持原排版便于检索文献）； ② 页眉艺术字、版权小字等零散内容——模型判断无需翻译时保留原样； ③ 日志若提示「N 段因网络失败」——重新点开始翻译即可补齐。

某一页排版效果不好？

把"回填"换成 reportlab 再试一次对比；仍不理想可把该页截图反馈。

浏览器没自动打开 / 界面打不开？

看那个黑色窗口里显示的网址（形如 http://127.0.0.1:8763/），手动复制到浏览器打开即可。 若窗口里有红色报错，把截图发给维护者。
实在起不来时，可双击 程序\备用-经典界面.bat 使用 经典桌面界面应急（功能与网页版相同，但不再更新、没有皮肤）。

想彻底离线免费用？

安装 https://ollama.com，执行 ollama pull qwen2.5:7b 下载模型，界面"服务"切到 Ollama 本地即可（质量略低于 DeepSeek，胜在免费离线）。

七、文件夹里都是什么？哪些能删？

位置是什么能删吗
启动PDF翻译.bat唯一入口，双击启动别删
使用说明.html本说明可删
程序\程序本体（代码、配置、术语库、模型）别删、别改名
程序\config.json你的设置（含 API Key，勿外传）删了要重填设置
程序\cache\翻译缓存可删；删后重译要重新计费
程序\models\版面分析模型（识别表格/图区）可删；删后该增强自动关闭，其余功能不受影响
程序\selftest_out\自测输出的对比图可删
程序\docs\开发文档（交接说明、任务清单）建议保留
